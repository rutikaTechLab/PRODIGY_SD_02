# 🎯 Number Guessing Game

> **A fun and interactive Python game where the computer chooses a random number and challenges you to guess it!** 🎲

---

## ✨ Features

- 🎲 Random Number Generation
- 📈 Too High / 📉 Too Low Hints
- 🏆 Attempt Counter
- 🔄 New Game Option
- 🎨 Modern GUI with CustomTkinter
- 🌙 Dark Theme
- ⚡ Fast & Responsive

## 🚀 Getting Started

### Install

```bash
pip install customtkinter
```

### Run

```bash
python guessing_game.py
```

---

## 🎮 How to Play

1️⃣ Enter a number between **1–100**  
2️⃣ Click **Check Guess**  
3️⃣ Follow the hints 📈📉  
4️⃣ Guess correctly to win! 🎉
